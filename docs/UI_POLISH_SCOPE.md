# TalentLens — UI Polish Scope (Slice 6B)

This phase is about consistency and visual polish.
NOT logic changes.

## In Scope

- Standardize card padding
- Align buttons horizontally
- Fix inconsistent font sizes
- Apply spacing system

- Clean up gap analysis layout
- Improve visual hierarchy
- Add subtle hover states

---

## Acceptance Criteria

- Every page uses same card system
- Every button uses defined variants
- No inconsistent padding
- No misaligned action buttons
- Typography consistent across app
- Landing page visually impressive
- Product UI clean and structured
