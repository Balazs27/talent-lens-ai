# TalentLens — Design Intent

TalentLens is an AI-powered HR intelligence platform.

The design must feel:

- Intelligent
- Calm
- Trustworthy
- Structured
- Professional
- Not playful
- Not flashy
- Not Web3
- Not crypto
- Not social media

It should feel like:
- Linear
- Vercel
- Stripe dashboards

Primary goal:
Make data readable and scannable.

The UI must prioritize clarity over decoration.
